This method will convert all values in your source-code, as long as they are wrapped in a markup that contains at least one of data-unit or data-unit-value, even if they have no value assigned. The value will then automatically assigned to the preferred unit system (which is by default imperial). If you have a value like 440 oz, it is automatically converted to 27.5 lbs to aid your visitors by offering more human-readable units.
